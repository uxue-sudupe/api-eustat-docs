Documentación Interactiva de la API de Eustat

El servicio API de Eustat permite acceder mediante peticiones URL a toda la información disponible en el banco de datos de Eustat. Las peticiones URL tienen la siguiente estructura:

https://www.eustat.eus/bankupx/api/v1/{lang}/DB/[id]

El campo {lang} se corresponde con el idioma. Es obligatorio y puede tomar los siguientes valores:

  • ES: español
  • EU: euskera
  • EN: inglés

El campo [id] es opcional y corresponde al identificador de la tabla con los datos o metadatos de interés:

  • Si no se incluye en la consulta, se obtiene el catálogo de tablas del banco de datos.
  • Si se incluye el id y se usa la función GET, se obtienen los metadatos de la tabla.
  • Si se incluye el id y se usa la función POST junto con la consulta en formato JSON, se obtienen los datos de la tabla.

Para formular las consultas en formato JSON, se puede usar la interfaz de usuario de la base de datos. En la página correspondiente a la tabla de interés, haga clic en el apartado “Disponer de esta tabla en su aplicación”. Esto le brindará información sobre la URL y la consulta que debe enviar para recoger los datos con la API.
